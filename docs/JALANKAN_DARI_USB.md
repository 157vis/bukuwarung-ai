# laris.AI — Jalankan dari USB / flashdisk

Ganti `F:` dengan huruf drive USB Anda (mis. `E:`, `G:`).

Project portable di **`F:\bukuwarungai`**.

---

## Cepat (double-click)

| File | Fungsi |
|------|--------|
| **`START-Dashboard.bat`** | Dashboard → http://localhost:8501 |
| **`START-Bot.bat`** | Bot lokal → http://localhost:8000 |

```powershell
F:
cd F:\bukuwarungai
.\scripts\run-dashboard.ps1
```

Lihat **`JALANKAN_DARI_USB.md`** di root USB untuk panduan lengkap + daftar tools.
